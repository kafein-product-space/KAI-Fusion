🤖 KAI-Fusion RAG Chatbot Widget Test

Widget Testi

Bu sayfa KAI-Fusion RAG chatbot widget'ının test edilmesi için hazırlanmıştır. Sağ alt köşede mavi bir chat butonu görmelisiniz.

Widget'ı test etmek için:

  1. Sağ alt köşedeki 💬 butonuna tıklayın
  2. Chat paneli açılacak
  3. Mesaj yazıp göndermeyi deneyin
  4. Demo response'u alacaksınız

🚀 Kolay Entegrasyon

Tek bir script tag'i ile herhangi bir web sayfasına entegre edilebilir.

📱 Responsive

Mobil ve desktop cihazlarda düzgün çalışır.

🎨 Özelleştirilebilir

Renk, boyut, pozisyon gibi görünüm ayarları data attribute'ları ile yapılabilir.

🧠 RAG Destekli

Doküman tabanlı akıllı yanıtlar verir.

Entegrasyon Kodu

Widget'ı kendi sitenize entegre etmek için şu kodu kullanın:

<script src="http://localhost:8002/static/widget.js"
        data-api-url="http://localhost:8002"
        data-api-key="YOUR_API_KEY_HERE" 
        data-label="💬" 
        data-color="#2563eb"
        data-width="720px" 
        data-height="540px"
        data-position="right" 
        data-open="false"></script>

Konfigürasyon Seçenekleri

  • data-api-url: RAG API sunucunuzun URL'i (zorunlu)
  • data-api-key: API anahtarınız (varsa)
  • data-label: Widget butonu etiketi (💬)
  • data-color: Widget rengi (#2563eb)
  • data-width: Panel genişliği (720px)
  • data-height: Panel yüksekliği (540px)
  • data-position: Pozisyon (right/left)
  • data-open: Başlangıçta açık olsun mu (true/false)

Domain Bazlı Cache

Her domain için ayrı cache tutulur:

  • datatouch.netfein.com → chat_datatouch.netfein.com
  • datafocus.netfein.com → chat_datafocus.netfein.com
